<p>
  🐍 Excepciones en Python son eventos que ocurren durante la ejecución de un programa 
  y que pueden alterar su flujo normal. 🚫 Las excepciones se utilizan para manejar 
  errores y condiciones excepcionales de forma controlada, permitiendo al programa responder 
  adecuadamente sin detenerse abruptamente.
</p>

Para ejecutar un programa, simplemente navega al directorio del programa y ejecuta el archivo Python correspondiente. Por ejemplo:

1. Ubicate en la ruta del programa:

     ```
    cd ruta-del-programa
    ```
2. Ejecuta el programa:

    ```
    python nombre-del-programa.py
    ```
